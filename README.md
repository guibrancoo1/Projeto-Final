# Projeto-Final

a) O objetivo deste projeto é simular o funcionamento de um interruptor de luz utilizado em uma casa, com um Arduino como unidade de controlo.
Neste projeto pretende-se que, através de um botão de pressão, seja possível ligar e desligar uma luz, representada por um LED, mantendo o estado atual mesmo depois de o botão ser largado, tal como acontece num interruptor real de casa. 
Sempre que ocorre uma alteração de estado, um buzzer emite um som, que confirmando a ação.

b)  Este projeto tem muitos exemplos práticos da sua utilização como por exemplo o controlo de iluminação doméstica, que simula o funcionamento de um interruptor de luz; Também no controlo de equipamentos elétricos, como por exemplo ligar ou desligar pequenos dispositivos;  E por ultimo a utilização para projetos educativos, para ajudar na aprendizagem de programação e eletrónica com Arduino.

c) Eu escolhi este projeto, porque é algo que nós seres humanos usamos muitas vezes ao dia sendo em casa ou até mesmo no trabalho, é um projeto que respeita as as indicações do professor e que tem também muita linguagem utilizada e trabalhada em sala de aula.
